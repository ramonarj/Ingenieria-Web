Leeme: Firefighter space

Objetivo

Creación de una aplicación web que ofrezca a un cuerpo de bomberos concreto gestionar información de forma rápida teniéndola toda abordada en una misma web accesible solo con una cuenta de bombero.

Soporta los siguientes roles:

  • El administrador puede dar de alta bomberos, y tendrá acceso a toda la información del parque.
  • Un bombero tiene acceso a su información personal. También puede ver información de cómo están gestionadas algunos aspectos del parque. Además tiene la posibilidad de ver sus turnos y cambiarlos si es necesario. Por último, puede acceder a un apartado de chat donde tiene la posibilidad de comunicarse con otros bomberos.

Vistas

  • index - vista inicial. Permite especificar una clase, o hacer login (esquina superior derecha).
  • admin - vista de elección de rol. Permite a un usuario admin elegir si quiere entrar como administrador o como usuario normal.
  • user - vista principal después del login. Permite al usuario el acceso a las diferentes vistas: horarios, equipo, chat y perfil, además de poder hacer logout.
Los vistas citadas abajo tienen en común una toolbar en la parte superior que permite al usuario moverse entre ellas, y un botón que permite al usuario hacer logout.
  • horario - vista del horario. Permite al usuario ver el calendario de turnos que le corresponde.
  • equipo - vista del equipo el bombero. Permite al bombero ver el equipo que le corresponde.
  • chat - vista del chat entre bomberos. Permite a los usuarios enviarse mensajes entre ellos.
  • perfil - vista del perfil de usuario. Permite al usuario ver su perfil y modificar algunos de sus datos (y su foto de perfil).
